# tinydate [![Build Status](https://travis-ci.org/lukeed/tinydate.svg?branch=master)](https://travis-ci.org/lukeed/tinydate)

> A tiny, reusable date formatter.


## Install

```
$ npm install --save tinydate
```


## Usage

```js
const tinydate = require('tinydate');

tinydate('unicorns');
//=> 'unicorns & rainbows'
```
